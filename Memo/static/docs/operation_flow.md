- 管理画面ログイン
    - 条件をクリアすると誰でも管理画面に入れるため、荒らされる危険も含まれている。そのため、管理者(スーパーアドミン:開発者1人)の承認を必要とする。
    - ログインユーザーは、承認申請ボタンを押すと、管理者に通知メールが送信される。
    - 管理者がユーザーからのメールを確認し、管理画面に移動して承認(権限はスーパーアドミンのみ可能)
    - ログインユーザー管理者からの承認すると、メールの通知を受け、決済ページに移動し480円を支払う。
    - 1回の決済にAIの使用ポイント24pt加算、また利用可能時刻24時間が加算される。有効期限内であれば再決済でポイントや時間の補充も可能。
    - 現在はテストカード番号424244242424242(他の入力は自由)で決済可能で、無料でログインできるが、修了後は自身の従量課金制のトークンパスワードを使用可能にしておけば、数百円の売上にはなる。
    - 決済完了メールが届き、記載のトークンパスワードをコピーしてログイン。
    - ポイントは0になってもAIが使えないだけだが、期限が切れた次の操作でログイン画面にリダイレクトされる。
    - AI機能は各ページで使用可能
        - 記事解析スコアリング	6pt
        - 記事英語翻訳	4pt
        - 固定ページ生成 / カテゴリーAI提案 / サムネイル生成	各2pt
    - 注意事項として、残り5pt以下または3時間以下になるとページ上部に警告が表示される。
    - ログアウトした際のトークンパスワードは利用期間中は同じものを使用する。

- マーケティング戦略
    - 評価モデル解説（アコーディオンで展開表示）
        - 評価目的：「翻訳コストを抑えながら最大の海外検索流入を生む記事を選定」するパレート型（20:80）の考え方
        - スコア評価モデル（100点満点）の4指標を説明：SEO検索ポテンシャル（40点）・技術普遍性（25点）・コンテンツ構造品質（20点）・拡散適性（15点）
        - 翻訳判定の閾値：80点以上で翻訳可能・75〜79点は候補・60点未満は対象外
    - スコア化済み記事一覧（AI解析済みの記事が一覧表示）
        - ダッシュボードの「AI解析」ボタンを先に実行しておくことで記事がスコア化される（6pt消費）
        - 総合点バッジ：スコアに応じて不透明度が変わり、視覚的に翻訳優先度がわかる
        - 内訳スコア（SEO/技術/構造/拡散）・推奨SEOタイトル・検索キーワード・評価理由が各行に表示
        - 80点以上の記事のみ「AI翻訳」ボタンが有効化（4pt消費）
        - 翻訳完了後はその行の直下に英語タイトル・本文が折りたたみ表示
- 固定ページ管理
    - グローバルナビ プレビュー：管理画面内で実際のナビバーをそのまま確認できるプレビューエリアを最上部に表示
    - グローバルナビ一覧 / フッターナビ一覧の2セクションに分けて管理
        - ナビ種別トグル（「グローバル」「フッター」ボタン）で切り替え後、プレビューに即時反映される流れを見せる
        - 表示/非表示トグルで一時的に非表示にしても記事ページ・テンプレートは残る仕様
        - ドラッグ&ドロップで順序を入れ替え（Sortable.js）、XHRで非同期保存、完了トーストが表示
        - 編集モーダル：タイトル・要約文・画像選択（セレクトボックス）・表示順を変更可能
        - 削除はDBのみ（テンプレートファイルは残る）
    - AI 固定ページ生成（2pt消費）
        - タイトル入力で Gemini AI が本文1000字・要約・キー（URL）を自動生成
        - 生成後はプレビューエリアに展開：キー / 画像（「画像を変更」でランダム入れ替え可） / 要約（編集可） / 先頭500文字
        - 「すぐにナビへ表示」スイッチと「保存後に投稿作成ページへ移動」スイッチで保存後の動作を選択
- ユーザーサムネイル管理
    - 3カラム構成（アップロード / AI生成 / サムネイル表示設定）を横並びで表示
    - アップロード：ユーザー選択（または追加のみ）＋ファイル選択で通常アップロード。3桁連番（001.png〜）で自動命名
    - AI生成（2pt消費）：割り当て先ユーザーを選択し「AI生成」ボタンを押すと Gemini Image Generation でアイコンを生成
        - 生成後は円形クロップされた透過PNG がプレビュー表示、保存するとサムネイル表示設定グリッドへ動的追加
        - モデルは `gemini-2.0-flash-exp-image-generation`（無料枠・同じGOOGLE_API_KEYを使用）
    - サムネイル表示設定：チェックボックスで表示ON/OFFを切り替え、ゴミ箱ボタンで削除マーク（バツオーバーレイ）→ 一括保存
        - チェック済みのサムネイルのみがユーザー編集画面の「サムネイル選択」の選択肢に反映される
    - 下部のユーザー一覧テーブル：現在のサムネイル（丸アイコン）・ファイル名を全ユーザー分確認可能
- カテゴリー管理
    - カテゴリー一覧：ID・カラー（色見本スウォッチ）・バッジ名・投稿数・登録日時を表示
        - 投稿数が1件以上のカテゴリーは削除ボタンが無効化（ツールチップで理由を表示）
        - 投稿数ゼロのカテゴリーのみゴミ箱ボタンで削除可
    - カテゴリー追加フォーム
        - カテゴリー名：英数字12文字以内。入力時にリアルタイムバリデーション（重複・文字種チェック）
        - カラー：カラーピッカーで選択。#666より暗い色のみ許容（バッジ可読性のため）
        - AI生成ボタン（2pt消費）：既存カテゴリーの名前・色を渡して「非重複名＋非競合色」を Gemini が提案、name・colorに自動セット
